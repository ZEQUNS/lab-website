# Laboratory Website Template

This is a simple static website template for a research laboratory, academic group, or university research team.

## Pages

- `index.html`: Home page
- `research.html`: Research areas
- `members.html`: Members
- `publications.html`: Publications
- `contact.html`: Contact
- `style.css`: Website styles

## How to Use

1. Open the folder in VS Code.
2. Double-click `index.html` to preview the website in your browser.
3. Replace all `XXX` text with your own laboratory information.
4. Edit the members, research areas, publications, and contact details.
5. Upload the files to GitHub Pages or your own web server.

## Suggested Content to Replace

- Laboratory name
- University and school name
- Research areas
- Member names and photos
- Publication list
- Contact email and address
